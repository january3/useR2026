# useR! 2026 meeting presentation

Here are the relevant links:

 * [bioshmods](https://github.com/bihealth/bioshmods) - A collection of R
   Shiny modules for visualization and display of bioinformatic data.
   Modules include so far:
    * gene tables and plots
    * heatmaps
    * gene set enrichments
    * "panel plots" - for overview of gene set enrichment analyses
    * volcano plots
    * discordance-concordance plots
    * PCA
    * data download
    * workflow info
 * [seaPiper](https://github.com/bihealth/seapiper) - A Shiny app based on
   bioshmods for visualizing and exploring RNA-seq data. Heavily used and
   mostly bug-free.
 * [SODAR](https://github.com/bihealth/sodar-server) - A platform for data
   and meta-data management; see [Nieminen et. al](https://academic.oup.com/gigascience/article/doi/10.1093/gigascience/giad052/7232111)
 * [KIOSC](https://github.com/bihealth/kiosc-server) - A platform for apps,
   basically a manager for docker containers.

The reporting part of bioshmods is still largely experimental; I would love
to get feedback and advice from you.
